# RELOJPREMIUM

Plugin de WordPress para gestionar las marcas de entrada y salida del personal.

## Instalación
1. Copia la carpeta `plugin` dentro del directorio `wp-content/plugins` de tu instalación de WordPress.
2. Activa el plugin **Reloj Premium HR** desde el panel de administración.

## Uso
- Emplea el shortcode `[rp_clock]` en cualquier página o entrada para mostrar los botones de **Marcar Entrada** y **Marcar Salida**. Solo los usuarios autenticados podrán registrar sus marcas.
- Desde el menú de administración aparecerá la opción **Reloj Premium** donde se listan los últimos registros.
